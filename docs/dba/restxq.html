2 Rest interface paths

    /dba

    • ALL
      Redirects to the start page.

    /dba/backup-create

    • GET
      Creates a database backup.

    /dba/backup-drop

    • GET
      Drops a database backup.

    /dba/backup-restore

    • GET
      Restores a database backup.

    /dba/backup/{$backup}

    • GET
      Downloads a database backup.

    /dba/database

    • POST
      Redirects to the specified action.
    • GET
      Manages a single database.

    /dba/databases

    • POST
      Redirects to the specified action.
    • GET
      Main page.

    /dba/db-add

    • POST
      Adds a resource.
    • GET
      Form for adding a new resource.

    /dba/db-alter

    • POST
      Renames a database.
    • GET
      Form for renaming a database.

    /dba/db-copy

    • POST
      Copies a database.
    • GET
      Form for copying a database.

    /dba/db-create

    • POST
      Creates a database.
    • GET
      Form for creating a new database.

    /dba/db-delete

    • GET
      Deletes resources.

    /dba/db-download

    • ALL
      Downloads a resource.

    /dba/db-drop

    • GET
      Drops databases.

    /dba/db-optimize

    • POST
      Optimizes the current database.
    • GET
      Form for optimizing a database.

    /dba/db-optimize-all

    • GET
      Optimizes databases with the current settings.

    /dba/db-query

    • POST
      Runs a query on a document and returns the result as string.

    /dba/db-rename

    • POST
      Renames a database resource.
    • GET
      Form for renaming a resource.

    /dba/db-replace

    • POST
      Replaces a database resource.
    • GET
      Form for replacing a resource.

    /dba/dir-change

    • ALL
      Changes the directory.

    /dba/dir-create

    • POST
      Creates a directory.

    /dba/file-delete

    • GET
      Deletes files.

    /dba/file-start

    • GET
      Starts a job.

    /dba/file-stop

    • GET
      Stops jobs.

    /dba/file-upload

    • POST
      Upploads files.

    /dba/file/{$name}

    • GET
      Downloads a file.

    /dba/files

    • POST
      Redirects to the specified action.
    • GET
      Files page.

    /dba/job-discard

    • GET
      Discards jobs.

    /dba/job-result

    • GET
      Downloads the result of a job.

    /dba/job-stop

    • GET
      Stops jobs.

    /dba/jobs

    • POST
      Redirects to the specified action.
    • GET
      Jobs page.

    /dba/log

    • POST
      Returns a log entry table.

    /dba/log-delete

    • GET
      Deletes database logs.

    /dba/log-download

    • POST
      Downloads database logs.

    /dba/login

    • ALL
      Login page.

    /dba/login-check

    • POST
      Checks the user input and redirects to the main page, or back to the login page.

    /dba/logout

    • ALL
      Ends a session and redirects to the login page.

    /dba/logs

    • POST
      Redirects to the specified action.
    • GET
      Logging page.

    /dba/logs-jump

    • GET
      Redirects to the URL that creates a log entry table for the specified timestamp.

    /dba/pattern-add

    • POST
      Creates a pattern.
    • GET
      Form for adding a new pattern.

    /dba/pattern-drop

    • GET
      Drops a pattern.

    /dba/queries

    • GET
      Queries page.

    /dba/query-close

    • POST
      Closes a query file.

    /dba/query-eval

    • POST
      Evaluates a query and returns the result.

    /dba/query-open

    • ALL
      Returns the contents of a query file.

    /dba/query-save

    • POST
      Saves a query file and returns the list of stored queries.

    /dba/query-update

    • POST
      Runs an updating query.

    /dba/resource

    • POST
      Redirects to the specified action.

    /dba/session-kill

    • GET
      Kills web sessions.

    /dba/sessions

    • POST
      Redirects to the specified action.
    • GET
      Sessions page.

    /dba/settings

    • POST
      Saves the settings.
    • GET
      Settings page.

    /dba/settings-gc

    • POST
      Triggers garbage collection.

    /dba/static/{$file=.+}

    • ALL
      Returns a file.

    /dba/user

    • POST
      Redirects to the specified action.
    • GET
      Returns a single user page.

    /dba/user-create

    • POST
      Creates a user.
    • GET
      Form for creating a new user.

    /dba/user-drop

    • GET
      Drops users.

    /dba/user-edit

    • POST
      Edits a user.

    /dba/users

    • POST
      Redirects to the specified action.
    • GET
      Returns the users page.

    /dba/{$path}

    • ALL
      Shows a "page not found" error.
